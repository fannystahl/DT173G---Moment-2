Moment 3 - CSS-preprocessorer och SASS

Syfte

Syftet med uppgiften är att:

  • Förstå syftet med CSS-preprocessorer.
  • Kunna använda SASS för att underlätta utveckling med CSS.
  • Kunna automatisera konvertering av SASS-filer med Gulp.
  • Vidareutveckla ditt start-paket för webbutveckling från tidigare laboration.

De nya kunskaperna kommer du sedan att kunna applicera i din egen projektwebbplats.

Steg 1 - automatisera konvertering av SASS till CSS

Vidareutveckla ditt repository för webbutvecklings-projekt från tidigare laboration.

Denna ska nu även kunna konvertera SASS (scss-filer) till vanlig CSS för publicering.

Det ska gå att skriva SASS-filer, som vid lagring ska konverteras till CSS som går att köra i webbläsare.

Tidigare funktionalitet (minifiering, konkatenering av CSS och JavaScript med mera) från tidigare laboration ska behållas, och fungera som tidigare.

Publicera dina färdiga filer till antingen Github eller Gitbucket.

Steg 2 - skapa en webbplats med SASS

Skapa en enklare webb-mall till en fiktiv webbplats med valfritt innehåll. Denna webbplats behöver endast bestå av en HTML-fil (men vill du lägga in flera är det ok).

Design, struktur etcetera bestämmer du själv.

Krav på innehåll i denna webbplats:

  • Överskrifter och text med valfritt innehåll.
  • En huvudmeny med minst fem meny-alternativ (kan länkas till enbart "#").
  • Ett färgschema av något slag, med minst tre stycken färger.
  • Responsiv webbdesign, både på innehåll och meny. Ska fungera väl på stora och små enheter.
  • Minst två stycken bilder av något slag.

Webbplatsens CSS-kod ska genereras från SASS-kod. Följande saker ska (minimum) finnas i din(a) .scss-fil(er):

  • Variabler.
  • Nesting.
  • Mixins.
  • Selector inheritance (nyckelordet "@extend").
  • If/else-satser.
  • Color functions (darken, lighten minimun, men använda gärna även fler)

Redovisning
